# Bone-Fracture-Classification-Using-Deep-Learning
A deep learning pipeline for automated bone fracture classification from multi-region X-ray images. Built on Google Colab with EfficientNetB4 transfer learning trained on 9,246 images. Achieves 98.02% test accuracy, 0.98 F1 score, and 0.9985 AUC score across fractured and not fractured classes.
